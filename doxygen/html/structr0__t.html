Detailed Description

ADF4351 R0.

Integer Value (INT) 16bits: [DB30:DB15]) Integer part of the feedback division factor.

Fractional Value (FRAC) 12bits [DB14:DB3] Numerator of the fraction input to Σ-Δ modulator. FRAC values from 0 to (MOD − 1) cover channels over a frequency range equal to the PFD reference frequency.

RF OUT = PFD × (INT + (FRAC/MOD))
