12 @par Description
13  * ESP8266 ILI9341 with multiple window support WEB server and other networking demos
14  * The specific display is an TM022HDH26 display I got from ebay.
15  * I wrote the following support functions from scratch
16  * ADF4351 Wideband Synthesizer diriver and test code
17  * Font compiler and display code
18  * ILI9341 display driver with multiple custom sized window support
19  * (I rewrote most of Adafruit code ili9341 graphics library)
20  * readPixel() and custom sized window scrolling
21  * Not limited by ILI9341 hardware scroll restrictions.
22  * CORDIC C table generator and 3D transformation code support functions
23  * Wire frame viewer and C generator for Earth coastline data
24  * I created two wireframe demos
25  * Cube wireframe dataset
26  * Earth coastline dataset - wireframe view still needs hidden line removal option
27  * Small Printf with full floating point support along with ftoa/atof and integer conversions
28  * BDF fonts and a BDF font to C conversion code and optimized display code
29  * WEB server using SD CARD with CGI processing - files and CGI results can be ANY SIZE!
30  * WEB server can update TFT display
31  * Simple door sign status update using web page - see html/msg.cgi and web/web.c for code
32  * Network server client example for display updates
33  * Uart network server client for serial uart to network bridge.
34  * Generic queue handling code
35  * Uart driver.
36  * POSIX wrappers for FatFS support by ChaN - provides unix file I/O operations
37  * POSIX time functions and RTC set with NTP
38  * Timers used by rtc and time functions
39  * HSPI code that can handle multiple chip select and clock frequencies
40  * Includes FatFS SD card support by ChaN
